# SEAPHOTOS

Uma galeria de fotos e vídeos virtual com sistema CRUD das mídias e dos perfis cadastrados no banco de dados.


## Tecnlogias

As tecnologias e linguagens utilizadas durante o desenvolvimento deste projeto, foram:

* PHP 8.3.3
* MySql 8.0
* JavaScript
* SQL
* HTML
* CSS

## Serviços

* Font Awesome
